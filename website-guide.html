Wie baut man eine Website?

Als ich im Sommer 2024 diese Website gebaut habe, stellte ich mir genau diese Frage. Anfänglich dachte ich, dass man den HTML-Code und CSS-Code für die Website manuell schreiben muss. Schon länger gibt es jedoch Programme, wie Webflow o.Ä., die ein benutzerfreundliches Designinterface haben und das Website-Bauen vereinfachen.

Schritt für Schritt  erklärt dieser Guide nun den Weg vom Designen der Website über das erstellen einer HTML-Datei bis zum Hosten der Website auf GitHub.
